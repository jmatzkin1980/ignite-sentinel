# Ignite Brief

Arguments received from the user invocation: `$ARGUMENTS`

Parse `PROJECT_ID` from:

```text
/brief PROJECT_ID
```

Run from the repository root:

```powershell
python -m sentinel /brief PROJECT_ID
```

Return the `02_requirements/project-brief.md` path.
