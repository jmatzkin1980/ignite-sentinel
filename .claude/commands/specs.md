# Ignite Specs

Arguments received from the user invocation: `$ARGUMENTS`

Parse `PROJECT_ID` from:

```text
/specs PROJECT_ID
```

Run:

```powershell
python -m sentinel /specs PROJECT_ID
```

If maturity is blocked, report the blocker instead of forcing generation.
