# Ignite Backlog

Arguments received from the user invocation: `$ARGUMENTS`

Parse `PROJECT_ID` from:

```text
/backlog PROJECT_ID
```

Run:

```powershell
python -m sentinel /backlog PROJECT_ID
```

Summarize generated epic, story, and acceptance criteria IDs.
