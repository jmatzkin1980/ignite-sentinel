# Traceability Model

Required lineage:

`RAW -> REQ -> GAP/DEC -> SPEC -> EPIC -> US -> AC`

Supported ID prefixes:

- `RAW`: raw input
- `REQ`: requirement
- `GAP`: gap or blocker
- `DEC`: decision
- `SPEC`: specification
- `EPIC`: epic
- `US`: user story
- `AC`: acceptance criteria
- `TC`: test case
- `CHG`: change event
